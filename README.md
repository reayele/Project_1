# Project 1 - UPC Validator

## Author
Rediet Ayele

## Description
Wrote a program that validates a 12-digit UPC-A code. The program asks the user for a 12-digit UPC, uses a function to calculate the correct check digit, and then tells the user if the full UPC is valid.

## How to Run
Run the Python file and enter a 12-digit UPC when prompted.
